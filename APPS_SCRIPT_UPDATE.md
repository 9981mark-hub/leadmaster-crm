# Google Apps Script 업데이트 가이드 (음성 파일 동기화)

모바일/PC 간 음성 파일을 동기화하기 위해, **구글 드라이브 업로드 기능**을 Apps Script 백엔드에 추가해야 합니다.

## 1. Apps Script 열기
1. 구글 스프레드시트에서 **확장 프로그램** > **Apps Script**를 선택하세요.
2. `Code.gs` (또는 `코드.gs`) 파일을 엽니다.

## 2. 코드 수정하기
(이미 코드를 붙여넣으셨다면 3번으로 넘어가세요)

## 3. ⭐️ 중요: 권한 승인하기 (이 과정을 꼭 해야 합니다!)
코드를 붙여넣고 저장만 해서는 구글 드라이브 접근 권한이 생기지 않습니다. **스크립트 편집기에서 한번 실행**을 시켜줘야 권한 요청 팝업이 뜹니다.

1. `Code.gs` 맨 아래에 다음 코드를 복사해서 붙여넣으세요.
```javascript
function debugAuth() {
  DriveApp.getRootFolder();
  console.log("권한 승인 완료!");
}
```
2. 상단 메뉴의 **저장(💾)** 버튼을 누르세요.
3. 상단 도구 모음에서 실행할 함수를 **`debugAuth`**로 선택하세요. (기본값은 `doGet` 등으로 되어있을 수 있습니다)
4. **[실행]** 버튼을 클릭하세요.
5. **"권한 검토(Review Permissions)"** 팝업이 뜨면 클릭합니다.
6. 구글 계정을 선택하고, **"고급(Advanced)"** > **"Go to ... (unsafe)"** > **"허용(Allow)"** 과정을 차례로 진행하세요.
7. 실행 로그에 `권한 승인 완료!`가 뜨면 성공입니다. (이제 `debugAuth` 함수는 지우셔도 됩니다.)

## 4. 배포 업데이트
1. 우측 상단 **배포** > **배포 관리** (또는 새 배포)를 클릭합니다.
2. **액세스 권한 (Who has access)**이 **"누구나 (Anyone)"**로 되어 있는지 꼭 확인하세요.
3. 설정이 맞다면 닫거나 업데이트를 완료합니다.
